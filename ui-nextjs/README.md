# ui-nextjs

Next.js SSR blog UI for bytecode.news using the public API contract in `docs/openapi.json`.

## Environment

- `BACKEND_SCHEME` (default: `http`)
- `BACKEND_HOST` (default: `localhost:8080`)
- `API_URL` (optional override, takes precedence)
- `PORT` (default: `3000`)

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```
